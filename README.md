# mini gps

a gps location device

## details

- 1 meters precision
- 1Hz fix update rate
- LiPo battery (12 hours screen on, 5 day standby)
- waypoint system

## hardware

- Adafruit ESP32-S3 Reverse TFT Feather
- Adafruit GPS Module (MTK33x9 Chipset)

## software

- FreeRTOS
- Adafruit GFX Library
- Adafruit GPS Library
